# Word Template Note

To create a proper Word template (template.docx):

1. Open Microsoft Word
2. Create a new document with:
   - Header with "Nagarro AgentiMigrate Platform" and logo placeholder
   - Predefined styles for:
     - Heading 1 (Arial, 18pt, Bold, Blue)
     - Heading 2 (Arial, 14pt, Bold, Dark Blue)
     - Normal (Arial, 11pt, Black)
   - Footer with page numbers and generation date
3. Save as template.docx in this directory

For now, the service will use default Pandoc styling if template.docx is not present.
